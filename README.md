# E-Commerce Demo (Flask + SQLite)

A simple e-commerce web application built using Python Flask and SQLite.  
Users can register, log in, browse products, add items to cart, and view a cart summary with total price.  
This project demonstrates backend development, database integration, authentication, and HTML templating.

---

## ✅ Features
✅ User Registration and Login  
✅ Secure Password Hashing using Passlib  
✅ Product Listing Page  
✅ Add-to-Cart System (Session Based)  
✅ Cart Total Calculation  
✅ SQLite Database  
✅ HTML + CSS Frontend  

---

## ✅ Tech Stack
| Component | Technology |
|----------|------------|
| Backend  | Python Flask |
| Database | SQLite |
| Password Security | Passlib (pbkdf2_sha256) |
| Frontend | HTML, CSS, Jinja2 |
